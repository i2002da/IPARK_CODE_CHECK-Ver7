⚙️ Google Drive 연동 설정 🔑 MASTER 전용

Google Drive에서 파일을 불러오려면 Google Cloud Console에서 발급한
API 키와 OAuth 클라이언트 ID가 필요합니다.
입력한 정보는 이 브라우저의 localStorage에 저장되며, 마스터 계정만 수정할 수 있습니다.

Google Cloud Console → API 및 서비스 → 사용자 인증 정보 → API 키
Google Cloud Console → 사용자 인증 정보 → OAuth 2.0 클라이언트 ID
승인된 JavaScript 원본에 이 페이지의 URL을 반드시 추가하세요:
Drive 폴더 URL의 마지막 부분: drive.google.com/drive/folders/폴더ID
