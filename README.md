# 📋 내역 코드 체크 시스템
**아이파크 건설 · 내역관리팀** | 버전 v8_r9

---

## 개요

원내역 파일의 자재·일위대가 코드를 자동으로 검토하고, 표준내역과 항목을 비교하는 웹 기반 도구입니다.
별도 서버·설치 없이 브라우저에서 HTML 파일 하나로 실행되며, 모든 데이터는 로컬 브라우저(localStorage)에만 저장됩니다.

---

## 실행 방법

```
CodeCheck_ipark_v8_r9.html  →  Chrome(권장) 등 최신 브라우저에서 열기
```

## 주요 기능

### 1. 공지사항

마스터가 공지를 작성하면, 모든 사용자의 로그인 시 화면 중앙에 팝업으로 표시됩니다.

**마스터 — 공지 작성/수정**
1. 헤더의 `📢 공지 관리` 클릭 (마스터 전용)
2. 제목(최대 80자) · 내용(최대 1000자) 입력
3. `공지 활성화` 토글로 표시 여부 설정
4. `저장 · 발행` 클릭 → 즉시 반영

**사용자 — 공지 확인**
- 로그인 시 활성 공지가 있으면 팝업 자동 표시
- `확인` 버튼 또는 팝업 바깥 클릭으로 닫기
- `오늘 하루 보지 않기` 선택 시 당일 재표시 안 함

> 공지 데이터: `cc_notice` / 오늘 숨김: `cc_notice_dismiss`

---

### 2. 서버 공유 파일 관리

마스터가 파일을 한 번 업로드해 두면 모든 사용자가 선택만으로 바로 사용할 수 있습니다.

**마스터 — 업로드**
1. 상단 `📤 파일 업로드 관리` 클릭
2. 파일 드래그&드롭 또는 클릭 선택 (`.xlsx` / `.xlsm` / `.xls`)
3. 파일 구분이 시트명 기준으로 자동 감지됨 — 필요 시 수동 변경 가능
4. `💾 저장` 클릭

**사용자 — 선택**
1. 각 업로드 카드의 `📂 서버 파일 선택` 클릭
2. 목록에서 파일 선택 → `선택` 클릭 → 즉시 로드

> 저장 용량 한도: localStorage 기준 최대 **45MB**

---

### 3. 코드 검토 (`▶ 코드 검토 실행`)

원내역 항목 코드를 자재리스트·일위대가리스트 DB와 대조합니다.

| 슬롯 | 파일 | 필수 |
|------|------|------|
| 🔵 원내역 | 검토 대상 원내역 | ✅ |
| 🟢 자재리스트 | 자재 코드 DB | 둘 중 하나 이상 |
| 🟡 일위대가리스트 | 일위대가 코드 DB | 둘 중 하나 이상 |

**결과 구분**

| 결과 | 의미 |
|------|------|
| ✅ 일치 | 코드·품명·규격 모두 일치 |
| ⚠️ 불일치 | 코드는 있으나 품명 또는 규격 다름 |
| ❌ 미등록 | DB에 없는 코드 |
| ➖ 코드없음 | 원내역에 코드 값 자체가 없는 항목 |

---

### 4. 표준내역 비교 (`⇄ 표준내역 비교`)

원내역 항목을 표준내역과 비교하여 포함 여부 및 수량 차이를 확인합니다.

| 슬롯 | 파일 | 필수 |
|------|------|------|
| 🔵 원내역 | 검토 대상 원내역 | ✅ |
| 🟣 표준내역 | 비교 기준 표준내역 | ✅ |

---

### 5. 결과 저장 (`⬇ 결과 저장`)

검토 결과를 Excel 파일(`.xlsx`)로 내보냅니다.

---

### 6. 사용자 관리 (마스터 전용)

헤더 `👥 사용자 관리` → 신규 계정 추가, 활성화/비활성화, 비밀번호 초기화

---

### 7. 비밀번호 변경

헤더 `🔐 비밀번호 변경` → 본인 변경 / 마스터는 타 사용자 강제 변경 가능

---

### 8. 라이트 / 다크 모드

헤더 `☀️ / 🌙` 버튼으로 전환. 설정은 브라우저에 저장됩니다.

## 파일 형식 요구사항

### 원내역
- 필수 컬럼: `코드` (또는 자재코드, 일위코드, CODE)
- 권장 컬럼: `품명`, `규격`, `단위`
- 인식 시트명: `원내역`, `내역`, `source`, `Sheet1` 등

### 자재리스트 / 일위대가리스트
- 컬럼: 코드, 품명, 규격, 단위
- 인식 시트명: `자재리스트`, `일위대가리스트`

### 표준내역
- 컬럼: 코드, 품명
- 인식 시트명: `표준내역`, `standard`

> 헤더 행이 1행이 아닌 경우 최대 10행까지 자동 탐색합니다.

## 데이터 저장 위치

모든 데이터는 해당 PC 브라우저의 localStorage에만 저장되며, 외부로 전송되지 않습니다.

| 항목 | localStorage 키 |
|------|----------------|
| 사용자 계정 | `cc_ipark_users` |
| 공지사항 데이터 | `cc_notice` |
| 공지 오늘 숨김 | `cc_notice_dismiss` |
| 공유 파일 목록 | `cc_srv_meta` |
| 공유 파일 데이터 | `cc_srv_data_*` |
| 테마 설정 | `cc_ipark_theme` |

> ⚠️ 브라우저 캐시·사이트 데이터를 삭제하면 계정, 공지, 파일이 모두 초기화됩니다.
